# Online Uno Game

__Description__: Multiplayer Uno Game in JavaScript

_Original Repo_: https://github.com/sfsu-csc-667-spring-2022-roberts/term-project-team-red-suns

### Demo Images




### Prerequisite

1. Install postgres
2. Create .env in this current dir and add the following line while filling int the required info from one of the devs
of this project

   
    DATABASE_URL=ASK_A_BACKEND_ENV_FOR_THE_URL


3. run


    npm install

4. run one of the below


    npm start
    npm run start:dev

5. http://localhost:3000/
